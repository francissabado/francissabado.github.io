Angry Prims: Interfacing with the Virtual World via Microsoft Kinect and SecondLife2014

Capstone

About Project

In this project we developed a three dimensional game similar to an existing popular two dimensional game Angry Birds in a virtual world like SecondLife that mimics physical properties such as gravity, force, and collision. We created a HTTP server that allows communication for a Microsoft Kinect client written in C# to smart objects in SecondLife. In addition, we developed sophisticated gesture detection in Microsoft Kinect that corresponds to actions and commands in SecondLife.
